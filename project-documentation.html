📋프로젝트 개요

TaskFlow는 노션 데이터베이스를 활용한 업무관리 웹 애플리케이션입니다.

  • 이메일 기반 회원가입/로그인 시스템
  • 관리자 승인을 통한 회원 관리
  • 노션 데이터베이스 연동 업무 관리
  • PWA 지원으로 모바일 친화적
  • 보안 강화된 인증 시스템

⚙️기술 스택

JavaScript ES6+ Node.js Express.js HTML5 CSS3 Notion API Supabase PWA Service Worker

🚀설치 및 실행 과정

1. 프로젝트 클론

git clone https://github.com/leoking9009/new.git cd new

2. 의존성 패키지 설치

npm install

설치되는 주요 패키지들:

  • @notionhq/client (v2.2.14) - Notion API 클라이언트
  • @supabase/supabase-js (v2.58.0) - Supabase JavaScript 클라이언트
  • express (v4.21.2) - Node.js 웹 프레임워크
  • dotenv (v17.2.2) - 환경변수 관리
  • live-server (v1.2.2) - 개발용 서버 (개발 의존성)
  • http-server (v14.1.1) - 정적 파일 서버 (개발 의존성)

3. 환경변수 설정

.env 파일을 생성하여 Notion API 키를 설정합니다:

NOTION_API_KEY=your-notion-api-key-here

4. 서버 실행

# 프로덕션 서버 (포트 4000) npm start # 또는 node server.js # 개발 서버 (포트 3000) npm run dev # 또는 npm run serve

5. 브라우저 접속

  • 프로덕션: http://localhost:4000
  • 개발 환경: http://localhost:3000
